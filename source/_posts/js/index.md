## 首先检查自己电脑是否安装了

> Mac和Windows系统一样
```
node -v

1、如果安装则会显示版本相关信息，如未安装则会显示如下

'node'不是内部或外部命令,也不是可运行的程序或批处理文件。

```

## 安装

进入官网 [node](https://nodejs.org/zh-cn/download/)选择相应的系统下载即可

![GitHub Logo](/public/images/node_down.png)

安装只需按照步骤进行即可，检查是否安装成功可按第一步重新走一遍即可
